RSA encryption decryption formulae

The formulae used are as follows :

  • n = P * Q
  • Φ = (P-1) * (Q-1)
  • e = i such that GCD(i,Φ)=1
  • d = (k*Φ+1) / e for some integer k
  • Cipher text = message^e mod n
  • Plain text = cipher^d mod n

Please refer to these formulae for understanding the underlying concepts.
